**Credit Scoring Model

** Project Overview**
[span_1](start span)This project was developed as part of the Machine Learning Internship at CodeAlpha[span_1](end span). [span_2](start span)The primary objective is to build a predictive model that can evaluate an individual's creditworthiness[span_2](end span)[span_3](start span)—classifying applicants into "Good Risk" or "Bad Risk" based on historical financial metrics[span_3](end span).

** Approach & Methodology**
[span_4](start span)Algorithm: Random Forest Classifier (Ensemble learning method)[span_4](end span).
[span_5](start span)Feature Engineering:Calculated a critical financial metric, the "Debt-to-Income Ratio", from the available financial records[span_5](end span).
[span_6](start span)Data Preprocessing: Implemented text feature mapping using `LabelEncoder` and standardized numerical variances using `StandardScaler`[span_6](end span).

** Model Performance & Evaluation Metrics**
[span_7](start span)The model was assessed using standard classification performance metrics[span_7](end span):
Overall Accuracy: 95%
[span_8](start span)ROC-AUC Score: 0.9953[span_8](end span)
[span_9](start span)Precision (Bad Risk): 93%[span_9](end span)
[span_10](start span)Recall (Bad Risk): 98%[span_10](end span)
